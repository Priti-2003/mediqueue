# ⚙️ Installation & Setup

## Clone Frontend

```bash
git clone https://github.com/Priti-2003/mediqueue.git
```

## Clone Backend

```bash
git clone https://github.com/Priti-2003/mediqueue-backend.git
```

---

# ▶️ Run Frontend

```bash
cd mediqueue

npm install

npm run dev
```

---

# ▶️ Run Backend

```bash
cd mediqueue-backend

mvn clean install

mvn spring-boot:run
```
